# Composant Studio: `table`

## Role

`table` represente des donnees structurees en lignes/colonnes (liste d enregistrements).

## Quand l utiliser

- Inventaire detaille,
- liste d elements repetes (objets, PNJ, effets),
- structures avec colonnes fixes.

## Ne pas confondre avec `row/column`

`table` = structure de donnees repetables.  
`row/column` = mise en page visuelle.

Si tu veux juste "mettre des blocs l un a cote de l autre", utilise `row` + `column`.  
Si tu veux stocker des lignes de donnees, utilise `table`.
