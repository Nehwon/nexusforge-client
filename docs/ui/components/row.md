# Composant Studio: `row`

## Role

`row` sert a organiser des blocs cote a cote (layout horizontal).

## Quand l utiliser

- Construire une mise en page visuelle (formulaire, panneau, colonnes de section).
- Aligner plusieurs `column` dans une meme ligne.

## Ne pas confondre avec `table`

`row` = layout visuel libre (presentation).  
`table` = donnees tabulaires repetables (enregistrements).
